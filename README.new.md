# egg-on-fire

## Install nodejs
https://nodejs.org/

## Dependencies Used
### Server
    - axios (Used to make HTTP Request)
    - body-parser (Parse request arguments)
    - cors (Cross Original Requests)
    - express (Routing framework)
    - mongoose
    - nodemon (So we don't need to restart the server to update code)
### Client
    - axios
    - moment
    - react-file-base64
    - redux
    - redux-thunk
